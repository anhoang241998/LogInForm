Cách 1: Clone về

Sau đó các bạn gõ câu lệnh: git clone ấn nút phải chuột paste link mới chép vô.

Cách này đơn giản hơn, các bạn chỉ cần nhấn vào góc phải màn hình có nút clone or download. Các bạn chọn Download ZIP
